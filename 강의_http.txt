http 모듈: 웹서버 구축을 위한 모듈

1. 요청 (Request), 응답(Response)
    : 사용자의 웹브라우저에 서버의 주소 입력 
        >> 웹페이지 출력 요쳥 "Request"  
            >> 요청받은 HTML 데이터를 전송 " Response 응답"

    - Request : 사용자의 요청
    - Response : 사용자의 응답

2. 헤더
    : 요청, 응답에 대한 정보가 담겨있는 부분

    - Request Header : 사용자가 요청한 주소
                        사용자가 요청한 데이터(GET,POST 등등의 데이터,COOKIE), 
                        환경, : 브라우저의 종류, 아이피 ....
                        전송방식 : method - GET, POST

    - Response Header : 서버가 출력하는 페이지에 대한 정보

3. 응답 코드
    
    요청 ->> 응답

    200 번대 코드 ->> 정상 응답
    300 번대 ->> 페이지 이동
        304 : cashe
            브라우저는 한번 다운로드 받은 웹 자원(.js,.css, 이미지, 비디오...)을 일정 시간동안 보관
            -> 동일안 요청 파일이 있는 경우 저장된 파일로 로딩.
            -> 성능이 개선
            
    400 번대 ->> 요청쪽(Request)에 이상이 있다.
    500 번대 ->> 응답하는쪽에서 이상(서버에 문제가 있는 경우)
